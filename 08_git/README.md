# Reproducible Data Science with Git and GitHub 
 
Monday, July 20
1:00 p.m. to 3:00 p.m.

This workshop introduces the fundamentals of version control using Git and GitHub and explains their role in reproducible and collaborative data science. Participants will learn how to track changes, document their work, and manage project history using Git within RStudio. 


# Preparing your computing environment
To complete the activities in this Decode Institute session, you must have Git, R, and RStudio installed on the computing environment you are using (your computer or CWIC). You will also need a github account. 
1.	[Git Install Instructions](https://git-scm.com/install/) 
2.	[R +  RStudio Install Instructions](https://posit.co/download/rstudio-desktop/#download)
3.	[Github account registration](https://github.com/signup?ref_cta=Sign+up&ref_loc=header+logged+out&ref_page=%2F&source=header-home)

   
When creating an account, keep in mind this is something you will likely use in the long term!  Here are a few recommendations:
1.	Use your real name - like "jane_doe" or "joseph_d"
2.	Keep it short - easy to type and remember
3.	Make it unique - so no one else has it
4.	Keep it professional - something you'd be okay with future colleagues or bosses seeing
5.	Don't use temporary information - avoid your school name or current city
6.	Reuse from other places - if you have other professional accounts (linkedIn, bluesky, ect), use the same name

If you are not able to install or register before the live Decode Institute session, don’t worry! The session will have live demos and the slides with instructions will available for you to follow along later!
